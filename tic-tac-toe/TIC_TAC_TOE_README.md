# Tic Tac Toe Web Game - ⚔️ vs 🛡️

A beautiful, responsive web-based Tic Tac Toe game with a Captain America theme built with HTML, CSS, and JavaScript.

## Features

- 🎮 **Two-player gameplay** - Players take turns placing ⚔️ Swords and 🛡️ Shields
- 🏆 **Score tracking** - Keeps track of wins for both players and ties
- 🎨 **Modern design** - Beautiful gradient background and smooth animations
- 📱 **Responsive layout** - Works on desktop and mobile devices
- ✨ **Visual feedback** - Winning combinations are highlighted with animation
- 🔄 **Easy reset** - Start a new game anytime

## How to Play

1. **Open the game**: Open `tic-tac-toe.html` in your web browser
2. **Take turns**: Players alternate placing ⚔️ Swords and 🛡️ Shields on the 3x3 grid
3. **Win condition**: Get three in a row (horizontal, vertical, or diagonal)
4. **Score tracking**: The game keeps track of wins for both players and ties
5. **New game**: Click "New Game" to reset the board and continue playing

## Game Rules

- ⚔️ Swords always start first
- Players take turns clicking on empty cells
- The first player to get three of their marks in a row wins
- If all cells are filled without a winner, it's a tie
- The game automatically detects wins and ends the current round

## Files

- `tic-tac-toe.html` - Main HTML file
- `tic-tac-toe.css` - Styles and animations
- `tic-tac-toe.js` - Game logic and functionality

## Technical Details

- Built with vanilla JavaScript (no external dependencies)
- Uses CSS Grid for the game board layout
- Responsive design with media queries
- Clean, object-oriented code structure
- Event-driven architecture

## Browser Compatibility

Works in all modern browsers:
- Chrome, Firefox, Safari, Edge
- Mobile browsers (iOS Safari, Chrome Mobile)

## Customization

You can easily customize:
- Colors by modifying the CSS variables
- Grid size by adjusting the CSS grid layout
- Animations in the CSS keyframes
- Game rules in the JavaScript logic
